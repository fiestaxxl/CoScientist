
# CoScientist
## Overview
This project focuses on building a system capable of analyzing scientific papers, specifically within the domain of chemistry, and providing answers to user questions based on their content. It involves parsing, processing, and storing paper data within a vector database (ChromaDB) for efficient information retrieval.

## Purpose
The primary purpose of this project is to create an intelligent assistant that leverages scientific literature to support research and knowledge discovery in the field of chemistry. It enables users to query and receive relevant information from scientific papers, both those available in a pre-existing database and those they provide themselves.

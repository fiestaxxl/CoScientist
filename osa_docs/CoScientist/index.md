
# CoScientist
## Overview
This module provides the core functionality for parsing and pre-processing scientific papers and generating executable code for scientific tasks. It handles the conversion of papers from PDF format to HTML, cleaning and structuring the HTML content, splitting it into semantic chunks, extracting images, and generating code based on task descriptions.

## Purpose
The purpose of this module is to enable the CoScientist project to ingest, understand, and execute tasks related to scientific literature. Specifically, it prepares paper content for storage and retrieval, and automates code creation for complex analytical workflows. This includes functionalities for preparing data for Large Language Models and executing generated code using either Groq or OpenAI models.

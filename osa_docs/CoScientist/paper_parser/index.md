
# Paper Parser
## Overview
This module is responsible for extracting and preparing content from scientific papers for analysis. It handles the conversion of papers from PDF format to HTML, cleaning and structuring the HTML content, and splitting it into semantic chunks suitable for further processing. Image extraction and encoding are also performed as part of this process.

## Purpose
The purpose of this module is to provide the core functionality for parsing and pre-processing scientific papers, enabling the project to ingest and understand the content of these documents. Specifically, it prepares the paper content for storage and retrieval via a vector database, supporting the development of an intelligent assistant capable of answering questions based on scientific literature. This includes functionalities for converting paper content into a structured format, extracting images, and preparing data for Large Language Models.

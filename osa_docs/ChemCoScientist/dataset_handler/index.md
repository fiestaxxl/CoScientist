
# Dataset Handler
## Overview
The Dataset Handler module provides functionalities for loading and filtering data from chemical datasets, specifically the ChEMBL database. It is designed to facilitate access to chemical and biological activity data for analysis and querying.

## Purpose
This module's primary purpose is to prepare and filter data originating from the ChEMBL database. Specifically, it enables users to select specific data columns and apply filtering conditions to obtain a focused dataset suitable for downstream tasks within the CoScientist project, such as building a knowledge base or answering scientific questions related to chemical compounds and their properties.

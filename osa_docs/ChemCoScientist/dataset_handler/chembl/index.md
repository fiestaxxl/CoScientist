
# Chembl
## Overview
The Chembl module provides functionalities for loading and filtering data from a ChEMBL dataset. It is designed to facilitate access to chemical and biological activity data for analysis and querying.

## Purpose
This module's primary purpose is to prepare and filter data originating from the ChEMBL database. Specifically, it enables users to select specific data columns and apply filtering conditions to obtain a focused dataset suitable for downstream tasks within the CoScientist project, such as building a knowledge base or answering scientific questions related to chemical compounds and their properties. The `ChemblLoader` class and `get_filtered_data` function are the core components responsible for achieving this data handling capability.

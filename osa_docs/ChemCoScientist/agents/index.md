
# Agents
## Overview
This module provides a collection of agents designed for diverse tasks within a scientific context, focusing on dataset creation, machine learning/deep learning model execution, chemistry-specific problem solving, nanoparticle analysis, and scientific paper analysis. These agents facilitate interaction with language models and specialized tools to achieve specific goals based on given tasks and configurations.

## Purpose
The primary purpose of this module is to encapsulate and manage the core logic for interacting with various specialized agents that form the foundation of the CoScientist project's analytical capabilities. It enables the automation of tasks such as dataset construction, model application, and domain-specific analysis, contributing to automated information extraction and knowledge discovery from scientific literature. Specifically, this module implements the functionality to build datasets, execute ML/DL models, perform chemistry-related calculations utilizing a Chemist agent, analyze nanoparticles, and to analyze scientific papers for information retrieval and question answering.

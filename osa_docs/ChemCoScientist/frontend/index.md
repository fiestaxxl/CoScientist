
# Frontend
## Overview
The Frontend module is responsible for the user interface and interaction logic of the CoScientist application. It handles displaying the chat interface, managing user inputs, processing uploaded files, and presenting results from the backend AI model. The module provides components for language selection, model configuration, and managing uploaded scientific papers and images.

## Purpose
This module serves as the primary entry point for users to interact with the CoScientist system. It focuses on presenting an intuitive interface for querying scientific data, uploading personal research papers, and receiving answers tailored to the content of those documents. The frontend orchestrates the communication between the user and the backend AI, managing the flow of information and ensuring a seamless user experience for performing analysis on scientific papers and obtaining insights. The module manages session state, user data directories, and the display of intermediate processing steps and generated outputs.

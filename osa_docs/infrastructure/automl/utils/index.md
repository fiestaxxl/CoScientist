
# Utils
## Overview
This module provides a collection of functions and classes designed to support automated machine learning (AutoML) workflows and the calculation of molecular properties. It focuses on data preparation, model training & prediction execution, and state management. The module also includes tools for evaluating molecular characteristics relevant to drug discovery and chemical analysis.

## Purpose
The primary purpose of this module is to encapsulate the core functionalities required for running AutoML pipelines, specifically tailored for molecular data. This includes preparing data from various sources (CSV files, SMILES lists), executing training and prediction processes using models like Fedot, and managing the state of these processes. Additionally, it provides a suite of functions for calculating key molecular properties to assess the quality and characteristics of chemical compounds. 

This module directly supports the project's goal of leveraging scientific literature by enabling the analysis and prediction of properties related to molecules discussed in research papers, allowing for deeper insights and more effective information retrieval.

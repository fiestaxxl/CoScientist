
# AutoML
## Overview
This module provides a collection of functions and classes designed to support automated machine learning (AutoML) workflows and the calculation of molecular properties. It focuses on data preparation, model training & prediction execution, and state management. 

## Purpose
The primary purpose of this module is to encapsulate the core functionalities required for running AutoML pipelines, specifically tailored for molecular data. This includes preparing data from various sources (CSV files, SMILES lists), executing training and prediction processes, and managing the state of these processes. Additionally, it provides tools for interacting with a remote server for model training and prediction, enabling the analysis of scientific papers by predicting properties related to molecules discussed within them, and facilitating more effective information retrieval across a vector database.

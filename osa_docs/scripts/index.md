
# Scripts
## Overview
The `scripts` module offers practical examples of how to interact with the core components of the CoScientist system. It contains example scripts demonstrating the utilization of dataset builder agents and Machine Learning/Deep Learning agents to process queries and retrieve results.

## Purpose
This module serves as a testing and demonstration ground for CoScientist’s functionalities. It showcases how to execute queries through agents and process the resulting responses. The module does not present core functionalities of the entire project but demonstrates how to utilize the CoScientist components in practice, facilitating understanding and experimentation with the system’s capabilities for analyzing scientific papers and answering user questions within the domain of chemistry.

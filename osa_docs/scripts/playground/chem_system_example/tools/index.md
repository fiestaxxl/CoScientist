## Tools Module: Core Chemical Interaction and Calculation
#### Overview
The `tools` module provides a comprehensive suite of tools for chemical property prediction, molecule manipulation, and representation. It incorporates functionalities for converting between molecule identifiers (names and SMILES), calculating molecular properties, visualizing molecules, and generating new molecules based on descriptive prompts. This module facilitates a range of chemical tasks, from basic molecular information retrieval to, more advanced property estimation.

#### Purpose
This module is designed to serve as a foundational chemical toolbox for the CoScientist project. It enables the analysis and understanding of chemical entities described within scientific literature, further supporting the project's goal of providing informed responses to user queries. Specifically, the functionalities within the tools module enable the conversion of a user’s chemistry-related question into a machine-interpretable format, enabling property calculation and predictions and assisting in molecule generation when needed.


# Pydantic Models
## Overview
This module defines the data structures used to represent responses, plans, actions, workers, chats, and translations within the CoScientist project. These models are used for structuring data exchanged within the system, representing different stages of processing and interaction.

## Purpose
The primary purpose of this module is to provide a clear and structured data representation for the core components of the question answering system. It defines the format for responses generated from scientific papers, plans for processing information, actions to be performed, worker definitions for task execution, chat interactions, and translation operations. These models facilitate data consistency and clarity throughout the CoScientist workflow, directly supporting the project’s goal of providing intelligent assistance based on scientific literature.

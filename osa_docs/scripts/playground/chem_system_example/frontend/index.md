
# Frontend
## Overview
This module encompasses the user interface and interaction logic of the application. It manages chat history, user input, session state, and provides functionalities for uploading files and images. It's designed to handle both English and Russian language settings and features example queries to guide users.

## Purpose
The primary purpose of this module is to provide a user-friendly interface for interacting with the scientific paper analysis system. It facilitates user queries, receives and displays responses (including text, intermediate thought processes, and visualizations), and manages user-provided data such as files and images. It also handles initialization of the application state, user preferences (language), and backend connections, enabling users to easily query and receive relevant information in a conversational manner. It also provides examples for user queries, and manages the user session data.

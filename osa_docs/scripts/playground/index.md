
# Playground
## Overview
The `playground` module provides example implementations demonstrating the interaction with core CoScientist components. It includes example scripts showcasing how to utilize dataset builder agents and ML/DL agents for processing queries and obtaining results. These scripts offer a practical demonstration of integrating with the broader CoScientist system.

## Purpose
This module serves as a testing and demonstration ground for CoScientist’s functionalities. It demonstrates on how to execute queries through agents and process resulting responses. The module doesn't present core functionality of the whole project, but show how to use the CoScientist components in practice. It facilitates understanding and experimentation with the system’s capabilities.
